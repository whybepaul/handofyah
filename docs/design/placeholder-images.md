## Usage Notes

1. **Unsplash License:** All images are free for commercial use. No attribution required, but crediting photographers is encouraged and good practice.

2. **Direct URL Parameters:** Append query parameters to direct URLs to control size and quality:
   - `w=1200&q=80` -- standard web display (recommended)
   - `w=800&q=80` -- product cards and thumbnails
   - `w=1920&q=85` -- full-width hero images
   - `w=400&q=75` -- small thumbnails and previews

3. **Image Sizing:** Use the `w` parameter rather than `h` to maintain aspect ratios. Unsplash CDN handles responsive sizing automatically.

4. **Replacement Plan:** These placeholders should be replaced with original Hand of Yah product photography as the brand develops. The aesthetic direction established here (warm, earthy, amber, botanical) should guide the photoshoot brief.

5. **Color Grading:** If images need slight adjustment to match the brand palette more closely, apply a warm color grade shifting toward terracotta/amber tones and reducing blue channel slightly.
